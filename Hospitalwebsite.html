ABC Hospital Website – Project Showcase

This website was created as part of my semester assignment for ABC Hospital. It was developed using HTML, CSS, and UIkit in Visual Studio Code, and includes features such as user login, appointment booking, doctor profiles, and department information, showcasing a fully responsive and functional website. For the backend, I used PHP for server-side scripting and MySQL as the database management system to handle data storage and retrieval efficiently.
